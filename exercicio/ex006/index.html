Formas x Sentido

Negrito / Destaque

Nesta frase, tem uma trecho em negrito usando a tag <b> (não semântico)

Nesta frase, tem uma parte em negrito usando a tag <strong> (semântico)

Itálico / Ênfase

Podemos colocar uma frase em itálico usando a tag "EM" (semântico)

Podemos também colocar a tag "I" (não semântico)

Marcação

Podemos marcar um texto com a tag <mark>

Textos Grandes e textos pequenos

Podemos deixar um texto grande e também pequena com as tag Big(obsoleta) e a tag small

Textos deletados e Textos Inseridos

Podemos deixar uma frase deletada usando a tag "DEL" e podemos deixar o texto sublinhado usando a tag "INS"

OBS: Podemos também deixar as frases sublinhadas usando a tag "U" (não semântica)

Palavras Subscrita e Sobrescrita

Podemos deixar uma frase subscrita usando a tag "SUB" e deixar sobrescrita usando a tag "SUP"
